# Rocky Resonance Board

A dependency-free mobile-first web app that plays two original Rocky-inspired synthesized voice modes from tappable tiles:

- `Alien Rocky`: resonant, musical, chord-forward communication
- `Human Rocky`: a speech-like translated interpretation

## Run

Open `index.html` in a browser or visit the GitHub Pages URL.

## Notes

- All tile labels are paraphrased, not direct quotes.
- Audio is synthesized in-browser with the Web Audio API.
- No backend or third-party voice cloning service is required.
